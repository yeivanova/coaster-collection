## Частная коллекция бирдекелей

Посмотреть сайт: [munyabrew.ru](https://munyabrew.ru/)

### Для запуска

- Склонировать репозиторий: git clone
- Установить пакеты: npm i
- Запустить проект: npm start

## Используемые технологии

- React
- Redux Toolkit
- React Router
- TypeScript
- Framer Motion
- Sass
- D3.js
